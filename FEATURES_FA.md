# مستندات کامل قابلیت‌های برنامه Spiritus

## نمای کلی
برنامه Spiritus یک اپلیکیشن مدیریت پنل VPN با قابلیت‌های پیشرفته است که به صورت خودکار تمام کانفیگ‌ها را پشتیبانی و پردازش می‌کند.

---

## 🔐 پروتکل‌ها و کانفیگ‌های پشتیبانی شده

### پروتکل‌های VPN
برنامه به صورت **خودکار** تمام پروتکل‌های زیر را تشخیص و پردازش می‌کند:

#### ✅ VMess (پروتکل V2Ray)
- **پشتیبانی کامل** از تمام تنظیمات VMess
- تشخیص خودکار از فرمت Base64
- پارامترهای پشتیبانی شده:
  - `add` - آدرس سرور
  - `port` - پورت (پیش‌فرض: 443)
  - `id` - UUID کاربر
  - `aid` - alterId (پیش‌فرض: 0)
  - `scy` - امنیت (auto/aes-128-gcm/chacha20-poly1305)
  - `net` - نوع شبکه (tcp/ws/h2/grpc/httpupgrade)
  - `path` - مسیر WebSocket/HTTP
  - `host` - هدر Host
  - `tls` - فعال‌سازی TLS
  - `sni` - Server Name Indication
  - `fp` - Fingerprint (پیش‌فرض: chrome)
  - `type` - نوع هدر (none/http)

#### ✅ VLESS (پروتکل مدرن V2Ray)
- **پشتیبانی کامل** از VLESS با تمام امکانات
- پارامترهای پشتیبانی شده:
  - UUID کاربر
  - `flow` - کنترل جریان (xtls-rprx-vision و غیره)
  - `security` - امنیت (none/tls/reality)
  - `sni` - Server Name Indication
  - `fp` - Fingerprint
  - `pbk` - Public Key (برای Reality)
  - `sid` - Short ID (برای Reality)
  - `type` - نوع انتقال (tcp/ws/grpc/h2/httpupgrade)
  - `path` - مسیر
  - `host` - هدر Host
  - `serviceName` - نام سرویس gRPC
  - `alpn` - Application-Layer Protocol Negotiation

#### ✅ Trojan
- **پشتیبانی کامل** از پروتکل Trojan
- پارامترهای پشتیبانی شده:
  - رمز عبور
  - آدرس و پورت سرور
  - `sni` - Server Name Indication
  - `fp` - Fingerprint
  - `type` - نوع انتقال
  - `path` - مسیر
  - `host` - هدر Host
  - `alpn` - ALPN
  - `security` - امنیت (پیش‌فرض: tls)

#### ✅ Shadowsocks (SS)
- **پشتیبانی کامل** از Shadowsocks
- تشخیص خودکار فرمت Base64
- پارامترهای پشتیبانی شده:
  - `method` - روش رمزنگاری
  - `password` - رمز عبور
  - آدرس و پورت سرور

#### ✅ Shadowsocks 2022
- پشتیبانی از نسخه جدید SS2022
- تولید خودکار کلید سرور
- روش‌های رمزنگاری پیشرفته

### انواع انتقال (Transport) پشتیبانی شده
برنامه **به صورت خودکار** تمام انواع انتقال را تشخیص می‌دهد:

- ✅ **TCP** - انتقال مستقیم TCP
- ✅ **WebSocket (WS)** - با پشتیبانی کامل از Path و Host
- ✅ **HTTP/2 (H2)** - انتقال HTTP/2
- ✅ **gRPC** - با پشتیبانی serviceName
- ✅ **HTTPUpgrade** - پروتکل جدید HTTPUpgrade
- ✅ **TCP با هدر HTTP** - برای عبور از فایروال

### امنیت و رمزنگاری
- ✅ **TLS** - پشتیبانی کامل با SNI و Fingerprint
- ✅ **Reality** - پروتکل جدید با Public Key و Short ID
- ✅ **مدیریت خودکار گواهی‌ها**
- ✅ **Fingerprint سفارشی** (chrome/firefox/safari/ios/android/edge/360)

## 🤖 پردازش خودکار کانفیگ‌ها

### تشخیص خودکار
برنامه **به صورت کاملاً خودکار** موارد زیر را انجام می‌دهد:

#### 1. تشخیص نوع پروتکل
```kotlin
✅ تشخیص از روی پیشوند:
   - vmess://  → پردازش VMess
   - vless://  → پردازش VLESS
   - trojan:// → پردازش Trojan
   - ss://     → پردازش Shadowsocks
```

#### 2. استخراج پارامترها
```kotlin
✅ پارس خودکار Base64
✅ پارس خودکار Query Parameters
✅ پارس خودکار JSON
✅ مدیریت خودکار URL Encoding
✅ مدیریت خودکار IPv4/IPv6
```

#### 3. ساخت کانفیگ Xray
```kotlin
✅ تبدیل خودکار به JSON
✅ ساخت outbound مناسب
✅ تنظیم streamSettings
✅ پیکربندی TLS/Reality
✅ تنظیم Transport Layer
```

#### 4. مدیریت خطاها
```kotlin
✅ بررسی اعتبار UUID
✅ بررسی فرمت آدرس
✅ بررسی پورت معتبر
✅ مدیریت پارامترهای اختیاری
✅ مقادیر پیش‌فرض هوشمند
```

### مثال پردازش خودکار

#### ورودی: لینک VLESS
```
vless://uuid@example.com:443?type=ws&security=tls&host=cdn.example.com&path=/ws#MyConfig
```

#### پردازش خودکار:
1. ✅ تشخیص پروتکل: VLESS
2. ✅ استخراج UUID
3. ✅ استخراج آدرس و پورت
4. ✅ پارس Query Parameters
5. ✅ تشخیص Transport: WebSocket
6. ✅ تشخیص Security: TLS
7. ✅ استخراج Host و Path
8. ✅ ساخت JSON کامل Xray

#### خروجی: JSON آماده
```json
{
  "tag": "proxy",
  "protocol": "vless",
  "settings": { ... },
  "streamSettings": {
    "network": "ws",
    "security": "tls",
    "wsSettings": { ... },
    "tlsSettings": { ... }
  }
}
```
